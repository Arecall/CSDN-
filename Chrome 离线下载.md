# Chrome 离线下载

###### 在网址最后加上：

```python
?standalone=1&platform=win64
```

###### 回车键后打开的是 64位下载页面，点击“下载Chrome”按钮，得到的就是 Google Chrome 64位离线安装包

###### 

```python
其中    ?standalone=1    指离线安装包，
```

###### platform=win64 指64位Windows 版本。

###### 下载32位的Chrome离线包

